**Pins_setup** est la partie front-end, que j'ai créé pour  exposer les  différents  images de setup de pc que j'ai reussi à obtenir.

---

## Descritpion

Il regroupe les meilleures idées de setups de bureau que j'ai pu récupéré pour tous les **amoureux** de pc, qui cherchent à allier:

- **Confort**
- **productivité**
- **choix d' éclairage**

Il existe aussi différents filtres pour retrouver ce que l'on veut par recherche de mots-clé, ou par filtre de catégories.
